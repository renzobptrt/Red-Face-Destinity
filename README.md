# Red-Face-Destinity
2D video game about a player who has to run as much as he can on randomly generated platforms.

## Characteristics
- Type: 2D
- Point of view: Second person
- Gender: Platform endless runner
- Platform: Mobile
- Graphics engine: Unity 2D
- Main language: C#
- Level : 1
